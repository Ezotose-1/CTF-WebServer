What is a CTF?

Capture the Flag (CTF) in computer security is an exercise in which "flags" are secretly hidden in purposefully-vulnerable programs or websites. Competitors have to find all flags hidden in a website by bypassing the security of the system and using a maximum vulnerability they can.
